# 💰 Controle Financeiro em Java

## 📌 Sobre o projeto
Sistema de controle financeiro desenvolvido em Java, com foco em Programação Orientada a Objetos (POO).  
Permite ao usuário registrar, visualizar e calcular gastos de forma simples via terminal.

---

## 🚀 Funcionalidades
- ✅ Adicionar gastos
- 📋 Listar gastos cadastrados
- 📊 Calcular total gasto

---

## 🛠️ Tecnologias utilizadas
- Java
- Maven
- POO (Encapsulamento e organização de classes)
